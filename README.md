🌿 Greenden – Plant Buying Website

Greenden is a clean and responsive plant e-commerce website built using HTML and Tailwind CSS.
It focuses on simplicity, modern UI, and a smooth user experience.

---

🚀 Features

- 🌱 Modern and minimal UI design
- 📱 Fully responsive layout
- 🛒 Product listing page
- 📩 Contact page
- ⚡ Fast loading using Tailwind CDN
- 🎯 Beginner-friendly structure

---

📂 Project Structure

- "index.html" – Home page
- "product.html" – Product listing page
- "contact.html" – Contact page
- "img/" – Images used in the website

---

🛠️ Built With

- HTML5
- Tailwind CSS (CDN)

---

🎯 Purpose

This project was built to:

- Practice frontend development
- Understand Tailwind CSS styling
- Create a clean and responsive layout
- Build a portfolio-ready project
